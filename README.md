# Dylan Thomas — Web Developer Portfolio

Personal portfolio website for Dylan Thomas, West-Mec certified web developer.

## Live Site
Deploy to GitHub Pages at: `https://yourusername.github.io`

## Setup
1. Upload all files to a GitHub repository named `yourusername.github.io`
2. Go to Settings → Pages → Deploy from branch (main)
3. Your site will be live in ~2 minutes

## Contents
- `index.html` — Complete single-file portfolio (HTML + CSS + JS)

## Tech
- Pure HTML, CSS, JavaScript — no dependencies, no build step required
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations
- Contact form with validation
